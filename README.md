# F.R.I.D.A.Y. OS — Stark Industries AI

> *"Good to meet you. I'm F.R.I.D.A.Y."*

## Setup

1. Go to **Settings** (tap the arc reactor icon)
2. Paste your **Gemini API key** from [aistudio.google.com](https://aistudio.google.com)
3. Click **Save & Engage**

## GitHub Pages Deployment

1. Upload `index.html` to your repo
2. Go to **Settings → Pages → Deploy from branch → main → / (root)**
3. Your site will be live at `https://YOUR_USERNAME.github.io/REPO_NAME`
